# aws-xray-service-map ![Chrome Web Store](https://img.shields.io/chrome-web-store/v/ggjelhbpihojpnkdlhdbllkfmbljjkhd.svg)

AWS X-Ray Service map visualizer
* Nodes are coloured based on a colour scale from green to red depending upon number of transactions/minute.
* Hide nodes depending upon a regex pattern. Useful to hide nodes mapping to other environments
* Highlight edges from selected node.

Run Cmd + Shift+ X to trigger the extension or click on the extension icon. Ensure that you have set the options first.
